# Random Card Game

## Program in action:
https://youtu.be/UU31cP--v1Y

![Random Card Game - Screenshot](Random_Card_Game-screenshot.png)

Picks from 8 random cards from 0-7 for each player and adds or subtracts points based on odds, evens and doubles.
